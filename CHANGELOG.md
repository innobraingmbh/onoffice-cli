### Added
- `onoffice:search` command for searching entities with filters and pagination
- `onoffice:get` command for fetching single records by ID
- `onoffice:fields` command for listing available fields
- JSON output mode for AI agent integration
- Publishable configuration file
- Support for 14 onOffice entities
